# Finance Dashboard

Finance dashboard project with role-based backend APIs and React frontend.

## Project Structure

- `backend`: Express + MongoDB API
- `frontend`: React + Vite app

## Quick Start

1. Start backend

```bash
cd backend
npm install
npm run dev
```

2. Start frontend (new terminal)

```bash
cd frontend
npm install
npm run dev
```
